Application Name: Appointment Organizer Plus+
Purpose: This program is a GUI based desktop application for maintaining client appointments. It interfaces with a
MySQL database and allows the user to add, modify and delete both customers and their associated appointments.

IDE: IntelliJ IDEA 2021.2.2
JDK: 17.0.2
JavaFX: 17.0.2
TODO MySQL Connector Driver: 8.0.25

Directions: TODO The user must initially enter a valid username and password existing within the database. Upon authentication,
the user is able to navigate either the Customer or the Appointment interfaces, where they can view and update a list of all
customers and associated appointments. Additionally, the appointments can be filtered in several different ways and the application
provides a couple reporting functions.
